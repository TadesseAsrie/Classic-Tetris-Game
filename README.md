# classic Tetris Game

A modern, responsive implementation of the classic **Tetris** game built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. The game features smooth gameplay, responsive controls, score tracking, level progression, sound effects, mobile support, and a clean modular code structure. It is designed as a learning project for front-end game development without using any external libraries or frameworks.

---

##  Features

-  Classic Tetris gameplay
-  All 7 official Tetromino pieces (I, O, T, S, Z, J, L)
-  Piece rotation with collision detection
-  Move left and right
-  Soft Drop
-  Hard Drop
-  Ghost Piece Preview
-  Hold Piece feature
-  Next Piece Preview
-  Automatic line clearing
-  Level progression with increasing speed
-  Score and High Score tracking
-  High Score saved using Local Storage
-  Pause and Resume functionality
-  Restart Game
-  Mobile touch controls
-  Background music and sound effects
-  Dark and Light Theme
-  Smooth animations
-  Statistics panel
-  Combo and bonus scoring
-  Fully responsive design
-  Accessibility support

##  Game Controls

| Key | Action |
|------|--------|
| ← | Move Left |
| → | Move Right |
| ↓ | Soft Drop |
| ↑ | Rotate Piece |
| Space | Hard Drop |
| Shift | Hold Piece |
| P | Pause / Resume |
| R | Restart Game |
